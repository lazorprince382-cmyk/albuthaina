# Deploy the Al Buthaina website

This is a standalone static HTML/CSS/JavaScript project. It is separate from the AMCO source and can be deployed as a new Netlify site without a build command.

## Netlify

1. Sign in to Netlify and choose **Add new site**.
2. Drag the complete `Al Buthaina Relief Charity Organisation` folder into Netlify Drop, or connect a new Git repository.
3. If asked for a publish directory, use `.` (the project root).
4. Test the temporary Netlify URL before connecting a domain.
5. Add the new charity's domain only after approval. Do not connect the existing AMCO domain to this project.

## Pre-deployment checks

- Confirm the official spelling of the organization name.
- Confirm the WhatsApp number `+256 745 630 373`.
- Confirm current transfer recipient details directly with the organization; they are not stored in the public source.
- Obtain consent for publishing identifiable people in photos and videos.
- Check each video on mobile data and remove any that should remain private.
